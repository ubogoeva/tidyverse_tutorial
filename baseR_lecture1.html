R как калькулятор

40 + 2 # сложение
[1] 42
20 - 2 # вычитание
[1] 18
42 * 2 # умножение
[1] 84
42 / 2 # деление
[1] 21
2 ^ 4 # возведение в степень
[1] 16

Функции для арифметических операторов: log(), sqrt()

sqrt(16) # квадратный корень, от слова square root
[1] 4
log(2) # по какому основанию логарифм?
[1] 0.6931472
log(exp(1)) # по умолчанию натуральный логарифм, то есть основание `e`
[1] 1

Можно задать основание логарифма. Как это сделать, можно узнать, вызвав справку по функции
?log.

?log
log(8, 2) # логарифм от числа 8 по основанию 2
[1] 3
log2(8) # тоже логарифм от 8 по основанию 2
[1] 3
# есть также отдельная функция для десятичного логарифма
log10(1000)
[1] 3

Переменные

Поскольку R является императивным языком программирования, то переменные и оператор присваивания занимают центральную роль. В качестве оператора присваивания используется <- Можно набрать, используя комбинацию клавиш Alt - (альт минус) или Option - на Mac.

Попробуем создать переменные.

a <- 1
b <- 'Hello World'

Векторы

Почти все в R является вектором, даже числа являются векторами длины 1.

Важная черта векторов - одинаковый тип данных в нем -> проще проиллюстрировать на примерах
